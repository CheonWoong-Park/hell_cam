초록 (Abstract)

본 연구는 추가 장비 없이 웹캠 1대만으로 스쿼트 자세를 실시간 분석하고 교정 피드백을 제공하는 브라우저 전용 시스템을 제안한다. MoveNet 기반 2D 포즈 추정에 등속도 칼만 필터를 적용해 관절 좌표와 속도를 안정적으로 추정하고, 유한 상태기계로 스쿼트 동작 구간(phase)과 반복 횟수를 검출한다.

스포츠 바이오메카닉스 문헌에 근거해 단일 전측면 카메라에서 측정 가능한 2D 지표로 8가지 자세 결함을 동작 구간별로 감지하고, 반복마다 점수를 6축 육각형(레이더)으로 분해·시각화한다. 모든 연산은 클라이언트에서 수행되어 영상이 외부로 전송·저장되지 않는다.

연구 배경·목적

  • 잘못된 스쿼트 자세(무릎 외반, 과도한 상체 숙임 등)는 ACL·요추 부상 위험을 높인다.
  • 전문가 피드백은 접근성·비용 제약이 크고, 기존 모션캡처는 고가·다중 카메라가 필요하다.
  • 목표: 웹캠 1대 + 브라우저만으로 동작하는 저비용·프라이버시 보존형 실시간 코칭.
  • 핵심 과제: 2D·단일 시점의 노이즈/모호성 속에서 신뢰할 수 있는 구간·반복·결함 검출.

주요 기여

  • 칼만 필터로 위치·속도 동시 추정 → 구간 검출 안정화(신뢰도 가중 측정노이즈).
  • 문헌 근거 8종 자세 결함의 2D 지표화 및 구간별 판정 규칙.
  • 반복 품질의 6축 분해 + 가중 평균 점수(점수와 육각형의 일관성 보장).
  • 영상 화면 위 실시간 큐 + 반복별 코칭의 이원화 피드백 UX.

관절 평활화 — 칼만 필터

관절 좌표 축마다 1D 등속도 칼만 필터(상태 = [위치, 속도])를 적용한다. 예측–보정 과정에서 위치와 속도를 함께 추정하므로, 구간 검출에 쓰이는 수직 속도를 별도 미분 없이 저지연·저잡음으로 얻는다.

  • 측정 노이즈 R을 keypoint 신뢰도로 스케일(R = R₀ / score²) → 흐릿한 관절은 덜 신뢰.
  • 큰 시간 간격(카메라 재시작) 시 자동 재시드.
  • 기존 EMA 대비 떨림↓ · 속도 추정 품질↑.

구간·반복 검출

엉덩이 깊이 비율·무릎 각도·수직 속도로 5개 phase를 판정하고, 히스테리시스 + 합법적 전이만 허용해 떨림을 억제한다.

  • 후보 지속시간(≥140 ms)·최소 반복시간·중복 쿨다운 게이팅.
  • bottom을 거쳐 standing 복귀 시 1회로 집계.
  • 무릎 각도 기반 판정으로 캘리브레이션 없이도 동작.

반복 점수 — 6축 육각형

score = depth·0.22 + knee·0.22 + posture·0.20 + balance·0.16 + tempo·0.10 + stability·0.10
− (치명적 오류 페널티)

부상 위험·핵심 축(무릎·깊이·상체)에 가중치를 높게 부여. 자세 결함은 해당 축을 직접 낮추므로 이중 감점을 피하고, 치명적 오류만 추가 감점한다.

한계 (Limitations)

  • 단일 2D 카메라 기반 → 정밀 3D 관절 각도·요추 굴곡(butt wink)·발 하중은 직접 측정 불가.
  • 조명·의복·카메라 거리·가림(occlusion)에 따라 keypoint 신뢰도가 변동.
  • 임계값은 문헌 근거의 2D 근사이며 개인 체형차 보정은 제한적.
  • 현재 스쿼트 단일 종목, 의료적 진단 목적 아님.

향후 연구

  • 전문가 라벨 대비 정량 평가(결함 검출 정밀도·반복 카운트 정확도).
  • MoveNet Thunder/멀티뷰·깊이 센서 융합으로 3D 확장.
  • 체형·장비 맞춤 캘리브레이션, 종목 확장(런지·데드리프트).
  • WebGPU 가속, 반복별 phase 타임라인 시각화.
